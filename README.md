# NJSUG Website

This repository contains the official website for the New Jersey SAS Users Group (NJSUG). The site is used to publish upcoming events, preserve historical meeting content, share organizational information, and highlight the NJSUG steering committee and community programs.

## Purpose

The website supports NJSUG's public presence by providing:

- Event pages for upcoming webinars and meetings
- Archive pages for past meetings, webinars, and conference materials
- Organization information such as the charter, contact page, and steering committee
- Search-engine-friendly metadata, sitemap, and structured content for key pages

This repository includes both current site content and legacy archive material. When updating pages, it is helpful to preserve historical accuracy for archived meetings while keeping current committee, event, and organization pages up to date.
